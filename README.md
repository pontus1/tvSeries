## How to run the app in a locally

### iOS

#### Requirements:
* Xcode and Xcode CLI
* Cocoapods
* Node version >12.x


#### Run:
Go to the root folder and run
```
npm run ios
```
or if you have yarn installed
```
yarn ios
```

### Android

#### Requirements:
* Android Debug Bidge (adb) or Android Studio
* Node version >12.x

#### Run:
Go to the root folder and run
```
npm run ios
```
or if you have yarn installed
```
yarn ios
```
